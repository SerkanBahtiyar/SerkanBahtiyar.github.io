Some Completed Projects

"Mobile Product Sales"
A program for the sale of local products has been implemented using Java and Firebase for Android programming.

"Dormitory Automation"
As the initial phase of the project, a section has been developed where information about students is collected, and transactions are carried out based on this information. In addition, the project includes transactions for staff members in the dormitory, tracking of the dormitory's expenses, and the use of structures such as stored procedures to manage and calculate operations related to the dormitory, such as keeping track of the number of occupants in rooms and adjusting the count when new students are added. This project has been developed using C# and SQL Server.

"Compiler Design"
A compiler design has been implemented using the Python language. The project includes the design of a compiler that handles variables, operators, mathematical operations (*, /, %, +, -, etc.), loops (for, while), conditional statements (if, else, elif), comment lines, and character recognition.

"City Introduction and Review/Complaint Project"
A project that introduces a city by showcasing places to visit, local dishes, and historical sites. It allows visitors to complain about aspects of the city they did not like, upload photos they took in the city, order local dishes, and leave comments on them. The project was developed using PHP, HTML, CSS, JavaScript, and MySQL.

"Commercial Automation"
This project involves a store's sales products and consists of three main parts: admin and user sections. The project is not limited to CRUD operations but also utilizes SQL structures such as Triggers and Procedures through Entity Framework and Linq queries, aiming to develop a comprehensive project. The ASP.NET MVC framework was used in the project.

"React-api-search"
A project was created using React and an API to fetch images based on search criteria, making the images responsive and adaptable to different screen sizes.

"Model Training"
A suitable and optimized model was trained using Python, Keras, and TensorFlow to recognize items such as jackets, coats, and dresses.
